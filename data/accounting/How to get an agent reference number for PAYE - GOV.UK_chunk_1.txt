Guidance How to get an agent reference number for PAYE Find out how to get an agent reference number for PAYE, Construction Industry Scheme (CIS) or Employment Related Securities (ERS). From: HM Revenue & Customs (/government/organisations/hm-revenue-customs) Published 7 September 2022 Last updated 22 April 2024 — The PAYE (Pay As You Earn), Construction Industry Scheme (CIS) and Employment Related Securities (ERS) services are all linked in the HMRC online services account. If you get an agent reference number and enrol one of these services, the others will automatically be added to your account. If you remove any one of these linked services, they will all be removed from your account and you will have to add them again. What you’ll need To get an agent reference number for PAYE, CIS or ERS you need to apply for one online. You’ll need your: full name trading name (if different) telephone number email address Get an agent reference number Use the online service to g et an agent reference number (https://www.tax.service.gov.uk/agent-epaye-registration/start? _ga=2.29887871.1360856042.1658308030-843212265.1644333853). Online services may be slow during busy times. Check if there are any problems with this service (https://www.gov.uk/government/publications/get-an-agent-reference- number-for-paye-for-agents-service-availability-and-issues). When you’ve got your agent reference number Once you’ve got your agent reference number, sign into HMRC online services to enrol the tax services you want onto your account (https://www.gov.uk/guidance/how-to- enrol-a-tax-service-onto-your-hmrc-online-services-account).
