Guidance Find software suppliers for the Construction Industry Scheme (CIS) Find recognised suppliers and software for the Construction Industry Scheme online. From: HM Revenue & Customs (/government/organisations/hm-revenue-customs) Published 10 October 2014 Last updated 11 December 2024 — Documents Construction Industry Scheme (CIS): commercial software suppliers (/government/publications/construction-industry-scheme-cis-commercial-software- suppliers/construction-industry-scheme-cis-commercial-software-suppliers) HTML Details Internet filing enabled software and online forms are available from HMRC and commercial software suppliers. These can be used to file CIS monthly returns (CIS300) and make verification requests. HMRC accepts forms and returns filed using any of the products listed in this document. We provide a free product, ‘Online Tax Return — CIS’ that lets you file monthly returns (CIS300) online. Register for the Construction Industry Scheme (CIS) online service (https://www.gov.uk/use-construction-industry-scheme-online) to get access to the product. HMRC has tested all the features of this service. If you need help or support, contact the online services helpdesk (https://www.gov.uk/hmrc-online-services-helpdesk).
